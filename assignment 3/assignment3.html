Binary

Introduction

Binary system, which is a great invention invented by Leibniz, because of the invention of binary, human society has entered the information age, and calculators running in binary have solved most of our daily needs. In binary system we usually use two different symbols 1 and 0 to show information, this system is widely used in the computer area, modern computers and computer-dependent devices use binary, no matter what language we used to program, the underlying layers are stored in binary form, so no matter which programming language we learn, we still should understand the calculation of binary.

The history of binary

Leibniz

Leibniz is a German polymath active as a mathematician, philosopher, scientist, and diplomat, also he may have been the first computer scientist and information theorist.

Leibniz's father a professor of ethics at the University of Leipzig, died when Leibniz was six-year-old, leaving a private library to him. At the age of 12, he taught himself Latin and began to learn Greek. He entered the University of Leipzig at the age of 14. He completed his studies at the age of 20, specializing in law and general university courses, In 1666 he published his first book on philosophy, “De Arte Combinatoria”.

The modern binary number system was fully documented by Leibniz in the 17th century in his article Explication “de l'Arithmétique Binaire”, this system is a way to convert verbal logic statements into mathematical ones, for this system, he only uses zeros and ones.

Leibniz wrote his system in an article called “Explanation of the Binary Arithmetic” in 1703 and shows how to use zeroes and ones to represent numbers. Leibniz expressed how simple the code is in the article. He wrote, “All these operations are so easy that there would never be any need to guess or try out anything.” Leibniz realized that binary could be used for automated calculation, but he didn't build this machine until 1937 George Stieglitz built a binary adder using relays.

Alan Turing

Alan Mathison Turing, an English mathematician and logician, is known as the father of computer science and artificial intelligence. His invention of the Turing machine was important for the widespread use of binary.

The Second World War interrupted Alan's normal research work, and in the autumn of 1939 he was called up to the Foreign Office's Communications Office to do military work, mainly to break enemy codes. Because of the need to decipher, he participated in the development of the world's first electronic computer.

When Alan finished his service at the Foreign Office in 1945, he tried to revive his pre-war research in theoretical computer science and combine it with his wartime work to develop new computers.

Turing solves a most major problem, that is computer logic or language, because the computer can't think straight specific transaction or store specific affairs, Turing in binary, a machine can recognize, understand and storage of language, will reality concrete with pure digital abstract world together.

The advantage of binary

Easy to judge logically

Logic algebra is the theoretical basis of logic operations, binary only have two numerical code, which can coincide with the logic operation, for example, True or false, On or off, Yes or no, Enabled or disabled, and Initiated or uninitiated.

Easy to implement

When the computer working, voltage exists at each of the outputs, and the high and low voltage is converted into binary by analog-to-digital conversion, high voltage represents by 1, low voltage by 0, below 2.5 volts is considered a low-level, and above 3.2 volts is considered as high level. It means that we can convert an analog circuit into a digital circuit, and this is much easier to implement in the circuit.

Have anti-interference ability

For each data in binary, it has only high and low two states, when there is a certain amount of disturbance, it can still reliably tell if it's high or low, so we think to use binary to show data have a lot of advantages, for example, high reliability and strong anti-interference ability.

How Binary actually work

Compilation

When we write a program, we use the language that people can understand to programming it, but computer can't understand our language, so we are going to compilation it. For example, we write a simple program written in C programming language, which are named “Hi.c”, if we want computer under it and do what we want, it has to go through the following steps: Source file (hi.c) => Pre-processing (Hi.i) => Compilation (Hi.s) => Assemble (Hi.o) => Linker => Executable.

First of all, we should pre-processing it, the file will change form source file “Hello.c” to “Hello.i”. The function of pre-processing is precompiled statements and file inclusion processing. That is, the macro instruction replacement and inclusion file is placed in the file to be compiled, and the complete C program source file will be generated after completion.

Second, we are going to do compilation, the file will change form “Hello.i” to “Hello.s”. The function of compilation is compiled the file to generate “.s” suffix assembly language file, that is, the file is assembly language code, assembly is a low-level language, directly to the hard disk operation.

Third, we will assemble the file, it going to change from “Hello.s” to “Hello.o”. The function of assemble is assemble the assembly language file, the main call assembly processing program to complete the assembly, assembly is the process of generating binary machine code, the file after the assembly ends with.o, called the object file.

Last, we should do the linker, links the generated object file with a standard library and generates the final executable file. Its function is to module link multiple object files generated by assembly and reference library files, and generates a complete executable file. When linked, all object files are arranged in the appropriate locations in the executable program. At the same time, the library functions called by the program are also linked to the program from their respective function libraries.

ASCII

ASCII code is the most widely used character set and its encoding in computers, which are used to convert binary to words or some other characters. We know binary has two different states 1 and 0, but two bits of binary number can represent four states 00, 01, 10, 11. Keep doing this, we can image what will happen when we have seven bits of binary, we have 2^7 of states, which means that we have 128 states in total, those codes can be arranged as a decimal number from 0 to 127. ASCII code is a code that use seven bits of binary number to encode and represent 128 different characters.

For example, the word “A” is represented by 1000001, we can easily find it in an image just on the top, so when we do programming, the computer can understand what we wanna do by translating the word to binary.

we can also use this image to change binary to the octal numeral system or hexadecimal numeral system, or we can change the octal numeral system and hexadecimal numeral system to those 128 characters just the same as binary, even although octal and hexadecimal systems are not used by modern computers, they are still necessary because they are easier and less error-prone to human interaction than binary systems.
